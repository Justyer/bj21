BlackJack

状态计算
玩家计算：手牌
牌

Entry:
桌子，玩家，卡牌

Relation:
桌子包含两名玩家和11张普通卡牌，卡牌的状态分为 在p1手上/在p2手上/在牌库

Action:
发牌，要牌，计算结果

## 牌桌逻辑

- 游戏开始：修改table状态为游戏中，初始化牌组，确定谁先，发一张暗牌，发一张明牌
- 要牌：给玩家发一张牌，本回合只能要一张牌
- 不要牌：结束本回合，判断上回合是否不要牌
    - 要牌：换成对方回合
    - 不要牌：本轮结束，转到结算
- 结算：判断双方点数大小，规则如下
    - 双方均小于等于21时，数字大者胜，相等则平局
    - 一方大于21时，另一方胜
    - 双方均大于21时，数字小者胜，相等则双败
